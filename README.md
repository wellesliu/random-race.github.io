## About

Random Race turns decision-making into an entertaining experience. Add your options, watch colorful dinosaurs race, and let fate decide the winner!

**Features:**
- Animated dinosaur races
- Customizable racers and options
- Multiple themes (day, night, sunset)
- Ad-free experience
